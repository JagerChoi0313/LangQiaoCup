确定字符串是否包含唯一字符

### 题目描述

实现一个算法来识别一个字符串的字符是否是唯一的（忽略字母大小写）。

若唯一，则输出`YES`，否则输出`NO`。

### 输入描述

输入一行字符串，长度不超过 100。

### 输出描述

输出一行，若字符串的字符是唯一的，则输出`YES`，否则输出`NO`。

### 输入输出样例

#### 示例

> 输入

```txt
abd25+
```

> 输出

```txt
YES
```

### 代码分析

```cpp
#include <iostream>
#include <unordered_set>
#include <cctype>  // for tolower function
using namespace std;

bool hasUniqueChars(const string& str) {
    unordered_set<char> seenChars;

    // 遍历每个字符
    for (char c : str) {
        // 转换为小写字母
        c = tolower(c);

        // 检查字符是否已出现过
        if (seenChars.find(c) != seenChars.end()) {
            return false;  // 找到重复字符
        }

        // 将字符添加到已出现字符的集合中
        seenChars.insert(c);
    }

    return true;  // 没有重复字符
}

int main() {
    string input;
    cin >> input;

    if (hasUniqueChars(input)) {
        cout << "YES" << endl;
    } else {
        cout << "NO" << endl;
    }

    return 0;
}
```

### 1. 引入库

```cpp
#include <iostream>
#include <unordered_set>
#include <cctype>  // for tolower function
```

- **`#include <iostream>`**：这个库包含了输入输出流，用来处理控制台的输入输出。
- **`#include <unordered_set>`**：这是一个无序集合（哈希表）的实现，用来存储字符并检查是否出现过重复字符。`unordered_set` 中的查找、插入操作都是常数时间复杂度 O(1)。
- **`#include <cctype>`**：这个库提供了字符处理函数，比如 `tolower()`，用来将字符转换为小写。

### 2. `hasUniqueChars` 函数

```cpp
bool hasUniqueChars(const string& str) {
    unordered_set<char> seenChars;
```

- **`hasUniqueChars`**：这个函数的作用是判断传入的字符串是否包含唯一字符。
- **`unordered_set<char> seenChars;`**：声明了一个 `unordered_set` 类型的变量 `seenChars`，它用来存储已经遍历过的字符。我们用它来检查字符是否出现过。

#### 遍历字符串

```cpp
for (char c : str) {
    c = tolower(c);
```

- **`for (char c : str)`**：这个循环会遍历字符串 `str` 中的每一个字符。每次循环，`c` 就是字符串中的一个字符。
- **`c = tolower(c);`**：使用 `tolower()` 将字符 `c` 转换为小写字母。这样可以忽略字母的大小写差异。比如字符 `'A'` 会变成 `'a'`，而字符 `'b'` 不变。

#### 检查是否重复

```cpp
if (seenChars.find(c) != seenChars.end()) {
    return false;  // 找到重复字符
}
```

- **`seenChars.find(c)`**：这个方法用于在 `seenChars` 集合中查找字符 `c`。如果找到该字符，`find()` 会返回一个指向该元素的迭代器；如果找不到，返回 `seenChars.end()`。
- **`seenChars.find(c) != seenChars.end()`**：如果 `find()` 返回的不是 `end()`，说明字符 `c` 已经在集合中出现过，即字符串中存在重复字符。这时我们可以立即返回 `false`，表示字符串不包含唯一字符。

#### 将字符添加到集合

```cpp
seenChars.insert(c);
```

- **`seenChars.insert(c)`**：如果字符没有出现过，使用 `insert()` 方法将字符 `c` 添加到 `seenChars` 集合中。这样下次遇到相同字符时就可以判断出来。

#### 没有重复字符的情况

```cpp
return true;  // 没有重复字符
```

- 如果遍历完整个字符串都没有发现重复字符，那么函数就会返回 `true`，表示字符串的字符是唯一的。

### 3. `main` 函数

```cpp
int main() {
    string input;
    cin >> input;
```

- **`string input;`**：声明一个字符串变量 `input` 用来存储用户输入的字符串。
- **`cin >> input;`**：通过 `cin` 从控制台读取输入的字符串，存储在 `input` 中。

#### 调用 `hasUniqueChars` 函数

```cpp
if (hasUniqueChars(input)) {
    cout << "YES" << endl;
} else {
    cout << "NO" << endl;
}
```

- **`hasUniqueChars(input)`**：调用我们之前定义的 `hasUniqueChars` 函数，传入用户输入的字符串 `input`，返回 `true` 或 `false`。
  - 如果返回 `true`，说明字符串字符唯一，输出 `"YES"`。
  - 如果返回 `false`，说明字符串有重复字符，输出 `"NO"`。

### 4. 程序结束

```cpp
return 0;
```

- **`return 0;`**：表示程序正常结束。

### 代码执行流程

1. 用户输入一个字符串。
2. 程序通过 `hasUniqueChars` 函数遍历字符串中的每个字符。
3. 将每个字符转为小写并检查是否重复。
4. 如果字符串没有重复字符，输出 `"YES"`，否则输出 `"NO"`。

### 时间和空间复杂度

- **时间复杂度**：O(n)，其中 n 是字符串的长度。遍历每个字符，并对集合进行插入和查找操作，每次操作的时间复杂度是 O(1)，因此总的时间复杂度是 O(n)。
- **空间复杂度**：O(n)，最坏情况下（没有重复字符），我们需要存储所有的字符，因此空间复杂度是 O(n)，其中 n 是字符串的长度。
