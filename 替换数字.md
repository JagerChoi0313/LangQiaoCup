替换数字
卡码网题目链接(opens new window)

给定一个字符串 s，它包含小写字母和数字字符，请编写一个函数，将字符串中的字母字符保持不变，而将每个数字字符替换为number。

例如，对于输入字符串 "a1b2c3"，函数应该将其转换为 "anumberbnumbercnumber"。

对于输入字符串 "a5b"，函数应该将其转换为 "anumberb"

输入：一个字符串 s,s 仅包含小写字母和数字字符。

输出：打印一个新的字符串，其中每个数字字符都被替换为了number

样例输入：a1b2c3

样例输出：anumberbnumbercnumber

数据范围：1 <= s.length < 10000。如何用C++做







这道题其实很简单，本质就是 **遍历字符串 + 判断字符是否是数字**。

思路分两种：

------

## ✅ 方法一：直接遍历拼接（推荐）

### 思路

1. 遍历字符串 `s`
2. 如果当前字符是数字（`isdigit()`）
   - 拼接 `"number"`
3. 否则
   - 拼接原字符
4. 输出新字符串

------

### 🔹 C++代码

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s;
    cin >> s;

    string result;

    for (char c : s) {
        if (isdigit(c)) {
            result += "number";
        } else {
            result += c;
        }
    }

    cout << result << endl;

    return 0;
}
```

------

## ✅ 方法二：原字符串上修改（不推荐）

因为：

- 原来 1 个字符要变成 6 个字符
- 容易发生覆盖问题
- 需要额外空间

一般字符串扩容替换用方法一最安全。

------

### 2️⃣ 字符串拼接

```cpp

```

很好 👍 这题用**双指针**做，其实是一个非常经典的“扩容型替换”问题（和替换空格很像）。

因为：

- 一个数字字符 → 变成 `"number"`（6 个字符）
- 长度会变长
- 不能从前往后直接改（会覆盖）

所以我们要 **先算新长度 → 扩容 → 从后往前填**。

------

# ✅ 双指针核心思路

## 第一步：统计数字个数

假设：

- 原字符串长度是 `n`
- 数字个数是 `cnt`

每个数字：

- 原来占 1 个字符
- 现在要变成 6 个字符
- 多出来 5 个字符

新长度：

```
newLen = n + cnt * 5
```

------

## 第二步：resize 扩容

```cpp
s.resize(newLen);
```

------

## 第三步：双指针从后往前填

设置两个指针：

```
i 指向原字符串末尾
j 指向新字符串末尾
```

从后往前移动：

- 如果 s[i] 是数字 → 在 j 位置写入 "number"
- 否则 → 直接复制

------

# ✅ C++完整代码（标准双指针版）

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s;
    cin >> s;

    int n = s.size();
    int count = 0;

    // 统计数字个数
    for (char c : s) {
        if (c >= '0' && c <= '9') {
            count++;
        }
    }

    // 扩容
    s.resize(n + count * 5);

    int i = n - 1;                     // 原字符串末尾
    int j = s.size() - 1;              // 新字符串末尾

    // 双指针从后往前
    while (i >= 0) {
        if (s[i] >= '0' && s[i] <= '9') {
            s[j--] = 'r';
            s[j--] = 'e';
            s[j--] = 'b';
            s[j--] = 'm';
            s[j--] = 'u';
            s[j--] = 'n';
        } else {
            s[j--] = s[i];
        }
        i--;
    }

    cout << s << endl;

    return 0;
}
```

# 🚀 为什么必须从后往前？

如果从前往后：

```
a1b2
```

当你把 `1` 替换成 `number` 时：

```
anumberb2
```

原来的 `b`、`2` 会被覆盖或移动困难。

从后往前就不会。

# ✅ 代码

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s;
    cin >> s;                // ① 输入

    int n = s.size();        // ② 原长度
    int count = 0;

    // ③ 统计数字个数
    for (char c : s) {
        if (c >= '0' && c <= '9') {
            count++;
        }
    }

    // ④ 扩容
    s.resize(n + count * 5);

    int i = n - 1;           // ⑤ 指向原字符串末尾
    int j = s.size() - 1;    // ⑥ 指向新字符串末尾

    // ⑦ 双指针从后往前填
    while (i >= 0) {
        if (s[i] >= '0' && s[i] <= '9') {
            s[j--] = 'r';
            s[j--] = 'e';
            s[j--] = 'b';
            s[j--] = 'm';
            s[j--] = 'u';
            s[j--] = 'n';
        } else {
            s[j--] = s[i];
        }
        i--;
    }

    cout << s << endl;
    return 0;
}
```

# 🎯 核心理解点

### 为什么要倒着写？

因为：

- 扩容后，新空间在后面
- 从前往后会覆盖未处理字符
- 从后往前可以保证：
  - 原字符还没被覆盖
  - j 永远在 i 右边

# 🧠 一句话总结

> 双指针的精髓就是：
> “空间变大，从后往前填；空间变小，从前往后压。”

# 🔑 核心理解

1. `for (char c : s)` 统计数字 → 用于计算新长度
2. `s.resize(n + count*5)` → 扩容，为后续写入 `"number"` 留空间
3. 双指针 `i/j`：
   - `i` 读原始字符串
   - `j` 写扩容后的字符串
   - 倒着写数字，保证原字符不被覆盖

# ⚡ 总结一句话

> 双指针从后往前填 + 预先扩容，是**原地替换字符串长度变化**的标准模板。
