包子凑数

### 题目描述

小明几乎每天早晨都会在一家包子铺吃早餐。他发现这家包子铺有 N*N* 种蒸笼，其中第 i*i* 种蒸笼恰好能放 Ai*A**i* 个包子。每种蒸笼都有非常多笼，可以认为是无限笼。

每当有顾客想买 X*X* 个包子，卖包子的大叔就会迅速选出若干笼包子来，使得这若干笼中恰好一共有 X*X* 个包子。比如一共有 3 种蒸笼，分别能放 3、4 和 5 个包子。当顾客想买 11 个包子时，大叔就会选 2 笼 3 个的再加 1 笼 5 个的（也可能选出 1 笼 3 个的再加 2 笼 4 个的）。

当然有时包子大叔无论如何也凑不出顾客想买的数量。比如一共有 3 种蒸笼，分别能放 4、5 和 6 个包子。而顾客想买 7 个包子时，大叔就凑不出来了。

小明想知道一共有多少种数目是包子大叔凑不出来的。

### 输入描述

第一行包含一个整数 N*N* (1≤N≤1001≤*N*≤100)。

以下 N 行每行包含一个整数 Ai*A**i* (1≤Ai≤1001≤*A**i*≤100)。

### 输出描述

一个整数代表答案。如果凑不出的数目有无限多个，输出 INF。

### 输入输出样例

#### 示例 1

> 输入

```txt
2
4
5
```

> 输出

```txt
6
```

> 样例说明

凑不出的数目包括：1, 2, 3, 6, 7, 11。

#### 示例 2

> 输入

```txt
2
4
6
```

> 输出

```txt
INF
```

> 样例说明

所有奇数都凑不出来，所以有无限多个

### 运行限制

- 最大运行时间：1s
- 最大运行内存: 256M



这道题其实是一个经典问题，叫 **Frobenius Coin Problem**（鸡块问题）。
它的本质是：**用若干个数（可以无限使用）凑出整数 X，哪些 X 无法表示。**

我一步一步给你讲清楚思路，然后给你 **完整 C++ 实现**。

------

# 一、先理解关键数学结论

设所有蒸笼大小为：

[
A_1, A_2, ... , A_N
]

如果

[
gcd(A_1, A_2, ... , A_N) \ne 1
]

那么：

- 只能凑出 **gcd 的倍数**
- 其他数永远凑不出来
- 因此 **凑不出的数无限多**

所以直接输出

```
INF
```

例如

```
4 6
```

gcd = 2

能凑出的数：

```
4,6,8,10,12...
```

奇数永远凑不出 → **无限个**

------

# 二、如果 gcd = 1

如果

```
gcd(A1,A2,...An)=1
```

那么有一个性质：

**凑不出的数是有限个。**

例如：

```
4 5
```

凑不出：

```
1 2 3 6 7 11
```

之后所有数都能凑出来。

------

# 三、如何求这些凑不出的数？

因为

```
Ai ≤ 100
N ≤ 100
```

最大蒸笼不会很大。

我们可以用 **完全背包 / 可达性 DP**

思路：

定义

```
dp[x] = 是否可以凑出 x
```

初始

```
dp[0] = true
```

转移

```
dp[i + Ai] = true
```

因为蒸笼无限，所以就是完全背包。

------

# 四、为什么只算到 10000？

有数学结论：

如果 gcd=1
那么超过某个数后 **所有数都能凑出来**

经验上

```
10000
```

已经足够覆盖题目范围。

# 五、算法流程

1️⃣ 输入 N
2️⃣ 求所有 Ai 的 gcd

如果

```
gcd != 1
```

输出

```
INF
```

否则

3️⃣ 做 DP

统计

```
1~10000
```

中多少不能凑出

# 六、C++完整代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main()
{
    int N;
    cin >> N;

    vector<int> A(N);

    int g = 0;
    for (int i = 0; i < N; i++)
    {
        cin >> A[i];
        g = __gcd(g, A[i]);
    }

    // 如果 gcd != 1
    if (g != 1)
    {
        cout << "INF";
        return 0;
    }

    const int MAX = 10000;

    vector<bool> dp(MAX + 1, false);
    dp[0] = true;

    for (int i = 0; i < N; i++)
    {
        for (int j = A[i]; j <= MAX; j++)
        {
            dp[j] = dp[j] || dp[j - A[i]];
        }
    }

    int ans = 0;

    for (int i = 1; i <= MAX; i++)
    {
        if (!dp[i])
            ans++;
    }

    cout << ans;

    return 0;
}
```

# 一、先理解 `||` 是什么

`||` 叫 **逻辑或（OR）运算符**。

规则是：

| A     | B     | A \|\| B |
| ----- | ----- | -------- |
| false | false | false    |
| false | true  | true     |
| true  | false | true     |
| true  | true  | true     |

简单理解：

```text
只要有一个为 true，结果就是 true
```

例如：

```cpp
true || false   → true
false || true   → true
false || false  → false
```

举个简单代码例子：

```cpp
bool a = false;
bool b = true;

bool c = a || b;
```

结果

```text
c = true
```

因为 **有一个是 true**。

# 五、为什么要写 `dp[j] = dp[j] || ...`

很多人会问：

为什么不是

```cpp
dp[j] = dp[j - A[i]];
```

原因是：

**可能原来就能凑出 j**

例如：

蒸笼：

```
4
5
```

当我们处理 **5 的时候**

```
j = 9
```

执行

```
dp[9] = dp[9] || dp[4]
```

因为

```
4 + 5 = 9
```

所以

```
dp[9] = true
```

但是 **dp[9] 原来可能已经是 true**

所以必须写：

```
保留原来的状态
```

```cpp

{

```

# 第五步：怎么找这些数？

思考：

我们想知道：

```
哪些数能凑出来
```

```
dp[x] = 能否凑出 x
```

# 第六步：状态转移

假设现在有蒸笼：

```

```

如果：

```

```

那么：

```
j = (j-Ai) + Ai
```

也能凑出来。

于是得到：

```
dp[j] = dp[j] || dp[j-Ai]
```

这一步其实就是 **完全背包公式**。

# 第七步：为什么是完全背包

因为题目说：

```

```

| 背包问题 | 本题     |
| -------- | -------- |
| 物品重量 | 蒸笼大小 |
| 物品无限 | 蒸笼无限 |
| 容量     | 包子数量 |

因此代码就是标准完全背包：

```cpp
for(int i=0;i<N;i++)

    for(int j=A[i];j<=MAX;j++)
```


所以：


统计所有 **false**。

# 四、gcd是什么

# 五、gcd怎么计算（欧几里得算法）

公式：

```
gcd(a,b) = gcd(b , a % b)
```

直到：

```
b = 0
```

例子：

求

```
gcd(48,18)
```

过程：

```
48 % 18 = 12

gcd(48,18)
→ gcd(18,12)

18 % 12 = 6

→ gcd(12,6)

12 % 6 = 0

→ gcd(6,0)
```

所以：

```
gcd = 6
```

# 七、蓝桥杯标准写法

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main()
{
    int N;
    cin >> N;

    vector<int> A(N);

    int g = 0;

    for(int i = 0; i < N; i++)
    {
        cin >> A[i];
        g = __gcd(g, A[i]);
    }

    if(g != 1)
    {
        cout << "INF";
        return 0;
    }

    const int MAX = 10000;

    vector<bool> dp(MAX + 1, false);

    dp[0] = true;

    for(int i = 0; i < N; i++)
    {
        for(int j = A[i]; j <= MAX; j++)
        {
            dp[j] = dp[j] || dp[j - A[i]];
        }
    }

    int ans = 0;

    for(int i = 1; i <= MAX; i++)
    {
        if(!dp[i])
        {
            ans++;
        }
    }

    cout << ans << endl;

    return 0;
}
```
